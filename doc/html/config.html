Before you can actually start using kueue, you should go to kueue -> Configure kueue and set its options to fit your needs.

2.1. General¶

This section includes the general options

  • DB-Server defines the server for kueue to get the data from

  • Your login needs to be set to your innerweb/unity login

The rest of the options should be self explaining.

2.2. Monitor¶

This section includes the options for the Queue Monitor

Here you can set which queues you would like to monitor for new SRs. The custom name can be self-defined, the dropdown box will give you a list of available queues.

2.3. QBoss¶

This section includes the options for kueue's QBoss features

If you are qboss you can make your life easier with kueue's QBoss features. You can set a list of engineers that appear as assignees in the defails window of an SR.

Also, if you are using KDE/Kopete as IM client, you can have kueue automatically notify an assignee after assignment.

2.4. Stats¶

This section includes the options for the Statistics feature

The statistics feature is not complete yet, as you can see in the configuration dialog.

The team-stats stuff is not yet used at all, so don't fill it out :-)

2.5. Unity¶

This section includes the options for the Unity integration

  • Unity password needs to be set to your innerweb/unity password

  • Unity URL needs to be set to the URL for the SI client

  • Download Directory specifies the directory for Unity to download files to per default

  • Use DownloadDirectory/SRNUMBER... - if checked, files will automatically be downloaded in seperate directories per SR

  • Enable toolbar - enable the unity toolbar with all shortcuts accessible via icons

  • Automatically extract archives - the name says it all

  • Enable editing with an external Editor - see the feature description in the "Unity Integration" section for details. If you'd like to use this feature, the next 2 options need to be set as well.

  • Open download directory - select either the default filemanager or a custom filemanager to open the download directories.

    As an example, here is what I am using: kueue_yakuake
