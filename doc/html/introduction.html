Chapter 1. Introduction¶

kueue is an SR queue manager, queue monitor and custom Unity browser. It provides the following features:

  • Personal queue view/sorting

  • Queue monitor incl. Bomgar features

  • Statistics (CSAT, Closed SRs)

  • Customized webbrowser for Unity
