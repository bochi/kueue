Chapter 4. Queue Monitor¶

The Queue Monitor browser shows an overview about SRs that are currently in selected pseudo queues.

One can directly assign/take SRs from the context menu or the details window.

As you can see, one can select several queues to be monitored. Kueue will notify when a new SR appears in a queue.

To get more details about an SR/CR, just open the details table. For information on what is shown, please see the screenshot.

This icon indicates that the customer is waiting in Bomgar. You can see the current Bomgar queue in the details for that SR.

This icon indicates that either the "High Value" or "Crit Sit" flag is set for that customer.

There are some shortcuts one can use to filter queues

  • F1   Show only EMEA SRs

  • F2   Show only US SRs

  • F3   Show only APAC SRs

  • F4   Show all SRs
