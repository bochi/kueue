Chapter 3. Personal Queue¶

The Personal Queue browser gives an overview about the current SR queue for an engineer, sorted by last activity or age.

In the default view, it will show the SR number, brief description, status, age and last update.

By clicking either on the arrow on the right side of the SR box, or using the context menu one can open a table that shows additional information on the SR.

By default, leftclick on an SR will show a context menu, middle click will toggle the table (show/hide info) and right click will copy the SR number to clipboard. The mouse button order can be changed in the General settings section though.

There are some shortcuts one can use to filter/sort the queue and change view options

  • F1   Show only "Awaiting Novell Support"

  • F2   Show only "Awaiting Customer"

  • F3   Show all other SRs

  • F4   Show all SRs (no filter)

  • F5   Expand/Close all SR tables (show/hide additional info)

  • F6   Switch sort type (age/last activity)

  • F7   Switch sort order (ascending/descending)
