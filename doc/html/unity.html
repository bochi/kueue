kueue comes with an integrated webbrowser that tries to workaround some of the flaws one faces when using the Siebel SI client.

This chapter describes the features one can use in the Unity browser.

7.1. Shortcuts¶

Here is a quick overview about the shortcuts that can be used within the Unity browser

  • F1   Send Email

  • F2   Set Solution Suggested

  • F3   Set Schedule For Close

  • F4   Open filebrowser with the current SR's download dir

7.2. Multiple Tabs¶

Using kueue, one can open as many Unity tabs as needed in parallel. With other browsers, this can be achieved by blocking cookies, but that approach also has its downsides (some of Unity's features seem to rely on cookies). That is not the case with kueue – it uses a seperate cookiejar for each open tab.

A new tab can be opened either from the menu or using the button in the tab bar.

7.3. Easier navigation¶

Rather than using Unity's query function to navigate to an SR, one can simply copy the SR number to clipboard and middle-click anywhere in the Unity browser or on the tab – this will automatically navigate to the SR.

Also, one can directly open Srs from the context menu in the other (Personal/Monitor) browsers.

7.4. Set product family and product¶

Since clicking through the product/product family popup windows can be quite annoying, one can simply right-click either in the „Product Family“ or „Product“ input-box and select the product there (see screenshot). This will automatically fill in the information and save the SR.

7.5. Set solution suggested activity¶

After suggesting a solution to the customer, one can use CTRL-S to set the activity automatically.

7.6. Smart download manager¶

One can set a download directory in the settings and also have kueue put the files into seperate directories per SR automatically. Also, it will automatically extract the archives.

After downloading a supportconfig tarball, kueue can create an NSA report directly from the download manager using the „NSA“ button. NSA is not required for this at all (it's a pain to install on Linux anyway these days).

7.7. Navigate back from „points of no return“¶

Sometimes Unity gets to a point where returning to the SR is not possible – which is a pain if additional changes have to be done in the SR (ie. Changing the status). Best example is after writing an email using the „Reply“ button. Kueue has a „Go back to SR“ option in the context menu which tries to eliminate that pain.

7.8. Save images and NSA reports¶

When viewing an image in Unity it will try to save it as „start.swe“ or if viewing an NSA report it will not let you save it at all. This is fixed with kueue, just check the context menu on an image or NSA report.

7.9. Open textboxes in external editor¶

Kueue lets you open textboxes in an external editor for editing. This is useful when writing emails etc from within Unity. When the document is saved in the editor it will automatically be written back into the textbox. Also, it will save the files so one can use tools like grep to find specific emails etc later on.

7.10. Set SR to „Schedule For Close“¶

Customer stops replying at some point? OK, one may want to set an ultimatum for the customer and set the SR to „Schedule For Close“ in X days. This can easily be done using the „CTRL-C“ shortcut in the Unity browser when viewing the respective SR.

It will create the „Schedule For Close“ activity for you as well as set the correct status.
